PTZ Camera Controls

Mouse Controls:
Left-click and drag anywhere to pan and tilt the camera.
Use the scroll wheel to zoom in and out.

Direct Value Controls:
Click on any value (Pan, Tilt, Zoom) to type in an exact value.
Click and drag left/right on a value to decrease/increase it.
When editing a value, use up/down arrow keys to increment/decrement.
Use the arrow buttons to increment/decrement values precisely.

Keyboard Shortcuts

  • H Toggle Help panel
  • M Toggle Mode panel
  • R Reset camera position
  • W Tilt up
  • A Pan left
  • S Tilt down
  • D Pan right
  • + Zoom in
  • - Zoom out
  • P Edit Pan value
  • T Edit Tilt value
  • Z Edit Zoom value
  • ↑ Increment selected value
  • ↓ Decrement selected value

Modes

Inverted Tilt Mode
Moving mouse up tilts camera up.
Moving mouse down tilts camera down.
Axis Lock Mode
Use LEFT-CLICK to modify PAN axis only.
Use RIGHT-CLICK to modify TILT axis only.
